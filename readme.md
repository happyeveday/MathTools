## 实现方法        
### 不同数据类型的数据互通        
通过template和使用<type_traits>库中的common::type<A,B>返回进行运算的两个数据类型中更高阶的数据类型实现        
### 不同数据类型之间的数据计算        
通过定义，目前仅仅定义了+,-,*,/，后续会加入**,&,|,~等计算        
### 生成函数图像        
通过FunctionPlotter.cpp文件和matplotlibcpp.h引入python，再通过python3.8提供的numpy以及matplotlib进行图像绘制
